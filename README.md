# Directory Tree Generator

The latest version can be downloaded from the [Released](https://github.com/RebelliousCat/Directory_Tree_Generator/releases) section.

A minimal tool that generates a directory structure including folders and optionally files of the folder in which it is executed.
This is the 1.0 version, rebuilt from scratch with more options and personalization icons representing file types, file and folder sizes, and some filtering options.


## Features

- Generate a directory tree **from the current working directory**
- Interactive mode selection at launch
- **Two** selection modes:
  - Folders **and** files
  - Folders **only**

- Optional filters:
  - **Minimum size** threshold `Filter files/folders limit minimum size.`
  - **Maximum size** `Filter files/folders limit maximum size.`
  - **Maximum depth** `Limit how deep the tree goes.`

- **Automatic unique log names:** `*root_folder*_YYYY-MM-DD_HH-MM-SS`
- **Header info:** `Logs include parameters used and generation date/time.`
- **File type icons:** "text, images, audio, video, archive, executable, scripts, temp, lock, config" `Identifies file types and assigns a icon to it.`
- **Folder size calculation:** `Displays total size of folders.`
- Tree structure rendered using **ASCII connectors**
- Optional **file-type and folder icons**
- Root path **displayed in the log summary**
- Output exported to a **timestamped `.txt` log file**
- **Log header with generation parameters:**

  - Generation date and time
  - Selected mode (folders only / folders and files)
  - Icons enabled or disabled
  - Minimum size filter
  - Maximum size filter
  - Maximum depth
  - Root path
  - Counts and displays:

    - Total number of folders
    - Total number of files




## Usage

- Place the executable **in the target directory**
- Run the **`.exe`**
- Select the desired options
- The log file is generated **in the same directory**
